# PIT Results – EvoSuite-generated Tests

This directory contains PIT mutation testing results obtained by executing
**EvoSuite-generated test suites only**.

## Purpose
EvoSuite represents a **search-based automated test generation approach**
and serves as a strong baseline in mutation testing research.

## Contents (generated during CI)
- `index.html` – PIT HTML report
- `mutations.csv` – per-mutation results
- coverage statistics

## Configuration
- EvoSuite version: 1.2.0
- Search strategy: default
- Test framework: JUnit 4 (JUnit Vintage)
- Only EvoSuite-generated tests are included

## Usage in Paper
Reported as:
> **Search-based automated test generation baseline (EvoSuite)**
