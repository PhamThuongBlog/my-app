# PIT Results – LLM-generated Tests

This directory contains mutation testing results for test cases
**generated exclusively by a Large Language Model (LLM)**.

## Purpose
This suite evaluates the effectiveness of **LLM-based test generation**
without any human-written or traditional automated tests.

## Contents (generated during CI)
- `index.html` – PIT mutation report
- `mutations.csv` – mutation-level outcomes
- coverage metrics

## Configuration
- Test source: LLM-generated unit tests
- Prompt-based generation
- No human or automated (Randoop/EvoSuite) tests included

## Usage in Paper
Reported as:
> **LLM-only test generation effectiveness**
